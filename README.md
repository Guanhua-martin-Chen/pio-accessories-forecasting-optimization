# PIO Accessories Forecasting Optimization

This repository supports a UCLA MEng Data Science capstone project with Hyundai Mobis / Mobis Parts America.

The project focuses on PIO accessories forecasting, including sales data exploration, data cleaning, demand forecasting, and forecast evaluation.

## Current Scope

The first phase focuses on:

- loading and inspecting raw Excel workbooks
- cleaning messy sales data
- creating a standard analysis table
- performing EDA
- preparing data for demand forecasting
